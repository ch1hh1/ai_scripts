このリポジトリ内のスクリプトはMITライセンスとしています。

| :-                              | :-      | :-           |
| 改行テキストから新規レイヤー作成.jsx | テキストフレーム内の文字1行ごとに<br>新規レイヤーを作成します | 2023.10.04 |
